Om uppgiften

Denna uppgift var intressant då jag fick direkt jobba med en api jag skapat själv, väldigt liknande förra uppgiften men med mer utav en mellanhand imellan. Återanvände en stor del av html och css koden ifrån förra uppgiften då det fungerade perfect för syftet (uppgifterna handlar om typ samma sak bara olika sätt att utföra det).

Använde åter igen en mysql databas så kommer spela in en video då webhosting inte låter roligt. Och enkla fetch anrop för att undivka ajax och liknande metodiker, blir bara förvirrande desto mer bloat man introducerar.

Webbsidan fungerar genom ett antal form element som tar deras inehåll via enkel DOM-manipulation och skapar object utefter inehållet med namn och värden som reflekterar formfaktorn hos apin. Varefter dessa object skickas till apin och om inget går snett så skes respektive ändringar. Men om apin ger error medelanden så skrivs dessa ut på webbplatsen.

All in all så är hemsidan ganska simpel och det endra problem som jag kan fixa men inte pallar är att all javascript är kommenterad på engelska (får mer sinnesfrid utav det fast det är nog en dålig ide) medans html är på svenska... vilket leder till att error utmatningarna från servern e på engelska ... :/ Aja! hoppas det inte anses problematiskt

JUST JA! syfte, jo syftet är att upfylla uppgifts kraven så jag kan komma närmare att inte ligga efter!

-Mvh Max Gagner
